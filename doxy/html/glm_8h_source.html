155 /* glmVertexNormals: Generates smooth vertex normals for a model.
156  * First builds a list of all the triangles each vertex is in. Then
157  * loops through each vertex in the the list averaging all the facet
158  * normals of the triangles each vertex is in. Finally, sets the
159  * normal index in the triangle for the vertex to the generated smooth
160  * normal. If the dot product of a facet normal and the facet normal
161  * associated with the first triangle in the list of triangles the
162  * current vertex is in is greater than the cosine of the angle
163  * parameter to the function, that facet normal is not added into the
164  * average normal calculation and the corresponding vertex is given
165  * the facet normal. This tends to preserve hard edges. The angle to
166  * use depends on the model, but 90 degrees is usually a good start.
167  *
168  * model - initialized GLMmodel structure
169  * angle - maximum angle (in degrees) to smooth across
170  */
171 GLvoid
172 glmVertexNormals(GLMmodel* model, GLfloat angle);
